PETUNJUK


Perbedaan

"Peta Skematik" adalah peta yang memiliki karakteristik berupa garis lurus skematik yang menggunakan sudut tertentu seperti 45 derajat atau 90 derajat serta berfungsi untuk menampilkan informasi penggunaan seperti sistem transportasi. Peta skematik memiliki bentuk dan pola yang berbeda dari kenampakannya di permukaan bumi. Sedangkan "Peta Geometrik" adalah gambaran permukaan bumi sebenarnya yang ditampilkan dalam bidang datar dengan skala tertentu dan dilengkapi simbol untuk memberikan informasi penggunaannya. Web ini menampilkan peta jaringan jalur kereta api di wilayah Jakarta dan Banten dalam bentuk skematik dan geometrik sehingga kedua peta memiliki tampilan yang berbeda sesuai penggunaanya.

Peta Skematik

Peta skematik dalam web ini menampilkan peta skematik dari jaringan jalur kereta api di wilayah Jakarta dan Banten. Pengguna dapat mengetahui jalur Kereta api Commuter Line Jabodetabek (KCJ) dari warna jalur yang berbeda - beda yang dijelaskan pada legenda peta skematik ini. Setiap pengguna juga dapat melihat "Jadwal KCJ"

dengan mengklik salah satu jalur dari peta skematik ini kemudian mengunduh jadwal tersebut baik untuk perjalanan pergi maupun perjalanan pulang.


Peta Geometrik

Peta geometrik dalam web ini menampilkan peta geometrik dari jaringan jalur kereta api di wilayah Jakarta dan Banten sesuai dengan kenyataannya di dunia nyata. Petunjuk penggunaannya:

  • Pengguna dapat memilih layar yang ditampilkan dari legenda dengan menghilangkan atau memberi centang pada kotak legenda. Pada kotak legenda juga menampilkan informasi dari warna yang ditampilkan pada peta
  • Setiap pengguna dapat melakukan scrolling, zooming, dan panning dari dari layar muka peta
  • Titik - titik putih yang bergerak pada layar muka utama peta merupakan tampilan kereta yang bergerak sesuai dengan waktu perjalanan Kereta api Commuter line Jabodetabek (KCJ)
  • Grafik waktu yang terletak di bawah peta menunjukkan grafik perjalanan Kereta api Commuter line Jabodetabek (KCJ) dalam rentang waktu 24 jam sehari. Pengguna dapat menjalankan atau memberhentikan gerakan secara otomatis. Sekaligus pengguna dapat memilih waktu perjalanan yang ingin ditampilkan
  • Pada sebelah kiri peta terdapat fungsi analisis. Terdapat empat fungsi analisis yaitu jalur perjalanan, rata - rata jarak, kelompok stasiun dan kelas stasiun
  • Setiap pengguna dapat memilih salah satu pilihan nilai atau lebih dari masing - masing fungsi analisis yang terdapat pada fungsi analisis tersebut dan melihat hasilnya langsung dari muka peta
  • Jika pengguna ingin mengetahui ada tidaknya KCJ pada waktu tertentu, maka yang pertama dilakukan adalah memilih jalurnya, kemudian menentukan waktunya dan mengklik stasiun dimana pengguna berada

    • TENTANG


    Latar Belakang

    Infrastruktur transportasi secara tidak langsung mempengaruhi pelaksanaan pembangunan di dalam suatu negara dalam mendukung pengambilan keputusan. Kereta api merupakan sarana transportasi yang efektif dan efisisen dalam mendukung aktivitas masyarakat di Indonesia. Kereta api dapat menampung penumpang dalam jumlah banyak dan berpindah dari suatu daerah ke daerah lain dengan waktu tempuh yang relatif cepat. Peningkatan pelayanan jaringan jalan kereta api difokuskan dengan memberikan kemudahan untuk mengakses perjalanan kereta api melalui web. Informasi perjalanan kereta api ditampilkan dalam bentuk peta online interaktif jaringan jalur kereta api dilengkapi analisis visual spatio temporal. Analisis visual bertujuan untuk menggabungkan kekuatan pengolahan data dari manusia dan komputasi dimana keduanya bekerja sama secara grafis untuk tercapainya pemahaman oleh pengguna. Halaman web ini merupakan hasil penelitian yang dilakukan untuk memenuhi syarat tugas akhir perkuliahan yang berjudul "Analisis Visual Spatio-Temporal untuk Jaringan Jalur Kereta Api dalam Peta Online Interaktif (Studi Kasus: Jaringan Jalur Kereta Api di Wilayah Jakarta dan Banten)".

    Tujuan

    1. Menghasilkan peta tentang jalur kereta api berbasis analisis visual dan spatio temporal melalui media web secara interaktif
    2. Menampilkan peta perjalanan Kereta Api Commuter Line (KRL) Jakarta dengan informasi waktu jadwal perjalanan dan lokasi
    3. Mendukung peta - peta tentang jalur kereta api di wilayah Jakarta dan Banten yang telah ada
    4. Membuat peta online interaktif dengan tampilan menarik dan mudah diakses oleh masyarakat


    Target Pengguna

    1. Pengambil kebijakan yang dapat digunakan sebagai sarana untuk mendukung pengambilan keputusan
    2. Peneliti yang memiliki ketertarikan dengan bidang perkeretaapian, analisis visual, spatio temporal dan bidang web geospasial
    3. Masyarakat umum yang membutuhkan informasi lokasi dan waktu dari perjalanan Kereta Api Commuter Line Jabodetabek (KCJ) dengan tambahan fungsi analisis
